# 🔐 Random Password Generator 

A simple **Random Password Generator** built using Python.
This program generates strong passwords based on user input such as length, digits, and symbols.

---

## 🚀 Features

* 🔢 Custom password length
* 🔐 Option to include digits
* 🔣 Option to include symbols
* ⚡ Random and secure password generation

---

## 🛠️ Technologies Used

* Python
* `random` module
* `string` module

---

## 📂 Project Structure

```id="l4v4d8"
password-generator/

└── README.md      # Documentation
```

---

## ⚙️ How It Works

* User enters password length
* Chooses whether to include digits and symbols
* Program generates a random password using Python libraries

---

## ▶️ How to Run

1. Install Python
2. Run the program:

```id="2tsq1o"
python main.py
```
